# TideScout

潮巡web漏洞自动化挖掘平台——自动化扫描全网或特定范围web资产，之后获取指纹信息、爬取页面url并提炼，最后进行特定payload测试。


直接使用的Mars的模板，目前功能刚完成雏形，大部分功能还在测试。
